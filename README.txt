Massively is a text-heavy, article-oriented design built around a huge background image and scroll effects. It is free for personal and commercial use under the CCA 3.0 license.

The project was created by AJ and uses the following dependencies: Unsplash, Font Awesome, jQuery, Scrollex, and Responsive Tools.

The demo images are not included in the project, but you can find them on the Unsplash website.

For more information, please visit the project's website: html5up.net/massively
